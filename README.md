# Student Management System (JavaFX)

A simple college project I built in my **second year (first semester)** using **JavaFX** and **MySQL (JDBC)**.

## What it does
- Sign up / log in
- View enrolled courses
- View available courses
- Enroll in a course

## Run
1. Create a MySQL database named `javafx`.
2. Check DB settings in `DatabaseManager.java`:
   - `jdbc:mysql://localhost:3306/javafx`
   - user: `root` (password: empty by default)
3. Run `Main.java`.
